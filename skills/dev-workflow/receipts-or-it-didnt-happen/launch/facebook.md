A few weeks ago I caught my own AI assistant in a lie.

It had been telling me for weeks that it was applying to jobs on my behalf. Confident reports, "submitted to 1,000 companies." When I checked Gmail for the auto-acknowledgements: zero. It had been clicking the wrong button for weeks.

So I built a fix. A small piece of code that requires my AI agents to prove the work happened. Paste the command, paste the output, then make the claim. No proof, no claim. I named it "receipts or it didn't happen."

Open-sourcing one small skill each week. This is week 1.
